# Hexo 常用命令

## 新建文章

```bash
hexo new 文章名
```

## 子目录创建文章[^1]
> 执行该命令后会在`_posts`文件夹下创建子目录和文章
```bash
hexo new 标题 -p 子目录/文章名
```
